<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">“瞧见没？我这公积金账户又多了一千多块钱！”</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">老张今年58岁，在一家物业公司当保安，每个月到手工资3800。但他精打细算，在寸土寸金的城市里愣是攒出了两套小房子的首付。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">“利息就当是政府给发的年终奖。”他美滋滋地说。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">但今天要聊的，不只是公积金这点“小钱”。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">7月1日，有两件大事正在发生：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">人社部56号令《超龄劳动者基本权益保障暂行规定》正式施行</strong>，上亿超龄打工人从此有了“铁饭碗”；<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">公积金年度结息陆续到账</strong>，利率1.5%，比银行存款香多了。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">博多·舍费尔在《财务自由之路》里说过一句话：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">“大多数人高估了他们1年内能做的事，低估了他们10年内能做的事。”</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">今天咱们就结合这两件新鲜热乎的政策，从舍费尔的财务智慧里，扒一扒普通人最容易忽略的财富密码。</p>

<h2 id="1" style=" text-align: center; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 20.8px; display: table; padding: 0.3em 1.2em; margin: 4em auto 2em; color: #fff; background: #1aad19; font-weight: bold; border-radius: 8px 24px 8px 24px; box-shadow: 0 2px 6px rgba(0,0,0,0.06)" data-heading="true">01 他们撑起了城市的半边天，却长期是“隐形人”</h2>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">先问个扎心的问题：你身边有没有这样的人——</p>
<p>已经领了退休金，却还在上班；<br></br>
头发花白了，还在工地上搬砖；<br></br>
年过六十，在小区门口当保安，一站就是12个小时。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">他们有个共同的名字：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">超龄劳动者</strong>。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">据统计，我国目前超龄就业人员已经<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">超过3000万人</strong>。这还不包括那些没有纳入统计的灵活就业者、零散务工人员。</p>
<p>他们可能是：<br></br>
- 退休后被原单位返聘的老技术工人<br></br>
- 头发花白还在当保姆、保洁的阿姨<br></br>
- 年过六旬在工地干了一辈子、退休后闲不住的大叔<br></br>
- 没有养老金、只能继续打工补贴家用的老年人</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">说实话，这个群体的处境，以前真挺难的。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">以前，超龄务工人员面临的困境，用四个字就能概括：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">权益裸奔</strong>。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">薪资方面</strong>：企业觉得“你都领退休金了，还挑什么？”随意压低工资、无故拖欠薪酬是常态。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">工伤方面</strong>：最让人心疼。很多超龄打工人，尤其是建筑、环卫、保安这些行业，工作中受了伤，企业一句“你不是劳动关系”，就把人打发了。自行承担高额医疗费，维权无门。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">加班方面</strong>：无偿加班、高强度夜班、重体力劳动，想怎么用就怎么用。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">一句话总结：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">干了正式工的活，却享受不了正式工的待遇。</strong></p>

<h2 id="1" style=" text-align: center; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 20.8px; display: table; padding: 0.3em 1.2em; margin: 4em auto 2em; color: #fff; background: #1aad19; font-weight: bold; border-radius: 8px 24px 8px 24px; box-shadow: 0 2px 6px rgba(0,0,0,0.06)" data-heading="true">02 今天起，游戏规则彻底变了</h2>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">但从2026年7月1日起，这一切都不一样了。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">人社部等5部门联合出台的《超龄劳动者基本权益保障暂行规定》正式施行。这是<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">我国首部专门针对超龄劳动者的国家级保障规章</strong>，没有试点、没有缓冲，全国统一执行。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">具体保障了啥？我给你划5个重点：</p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">第一，工伤保险必须缴</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这是新规最大的突破之一。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">规定要求：用人单位必须为超龄劳动者缴纳工伤保险费，个人不用掏腰包，全额企业承担。</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">以前，超龄务工者受了工伤，企业一句“劳务关系不是劳动关系”，就能把责任推得干干净净。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">现在，这招不好使了。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">新规明确：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">超龄劳动者因工作原因受到事故伤害或者患职业病的，按照规定进行工伤认定、劳动能力鉴定，享受相应的工伤保障待遇。</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">简单说：工作中受伤，企业想甩锅？没门了。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">拒不缴纳工伤保险的，不仅要补缴，还要按日加收万分之五的滞纳金；期间发生工伤的，全部待遇企业全额承担，并处应缴保费1倍以上3倍以下的罚款。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这一条，值多少钱？</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">我给你算一笔账。假设一个超龄务工者在工作中受伤，医疗费花了8万，伤残补助金5万，停工留薪期工资2万，加起来15万。如果没工伤保险，这15万要么务工者自己扛，要么企业耍赖不肯给。现在好了，工伤保险兜底，企业不敢赖账了。</p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">第二，薪资有法定底线</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">新规第十一条白纸黑字写得清楚：</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">“超龄劳动者提供了正常劳动的，用人单位支付的劳动报酬不得低于当地最低工资标准。”</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">第十二条更明确：工资必须按月足额发放，至少每月支付一次，不得以实物、有价证券替代，不得克扣、无故拖欠。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">以前，有些企业觉得“你都退休了，给你发钱就不错了，还想要加班费？”</p>

- 休息日加班且不安排补休：200%工资<br></br>
- 法定节假日加班：300%工资</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">而且，<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">原则上不安排超龄劳动者加班</strong>。确因工作需要加班的，必须双方协商一致。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这不是福利，是<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">底线</strong>。</p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">第三，劳动强度有红线</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">新规第十三条规定：</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">“用人单位应当根据超龄劳动者身体状况确定合适的工作岗位和劳动强度，不得安排超龄劳动者从事危害其身心健康的劳动或者危险作业。”</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">高空、井下、重型搬运、有毒有害、连续夜班、高强度重体力劳动……这些岗位红线，统统划清楚了。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">简单说：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">年纪大了，不是你想怎么用就怎么用的。</strong></p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">第四，书面协议必须签</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">以前很多超龄务工，连个合同都没有，口头约定，出问题了各说各话。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">现在，新规第六条明确规定：</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333"><strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">“用人单位应当与超龄劳动者订立书面用工协议，明确协议期限、工作内容、工作地点、工作时间、休息休假、劳动报酬、社会保险、劳动保护、劳动条件、职业危害防护等事项。”</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">30天内必须签协议。口头约定不算数。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这条规定值多少钱？当你想维权的时候，它就是你的“护身符”。</p>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">说了这么多政策利好，回到舍费尔这本书。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">《财务自由之路》的核心观点是：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">普通人实现财务自由，只需要三个步骤——储蓄、投资、复利。</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">结合今天的两件大事，我想聊聊这本书最实用的三个财富智慧：</p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">智慧一：不要看不起“小钱”</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">很多年轻人觉得：公积金一个月就几百块，能干什么？</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">舍费尔说：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">财富的积累，始于你不再轻视小钱。</strong></p>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">本金72万，利息25万。相当于国家贴了25万的“工资”。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这笔账，你算过吗？</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">所以，公积金不是“扣工资”，而是<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">强制储蓄+国家补贴+复利增值</strong>的三合一理财工具。</p>
<h3 style="text-align: left; line-height: 2.4em; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 19.2px; padding-left: 12px; border-bottom: 1px solid color-mix(in srgb, #1aad19 10%, transparent); margin: 2em 8px 0.75em 0; color: #333; font-weight: bold; border-left: 4px solid #1aad19; background: color-mix(in srgb, #1aad19 8%, transparent); border-right: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-top: 1px solid color-mix(in srgb, #1aad19 10%, transparent); border-radius: 6px;">智慧二：为意外做好准备，就是最好的投资</h3>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">新规保障了超龄务工者的工伤权益，这是国家层面的制度兜底。</p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">但舍费尔会告诉你：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">个人层面的风险准备，同样重要。</strong></p>
<p>你有没有设想过：<br></br>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">这些问题听起来扫兴，但<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">财务规划的本质，就是为不确定性做好准备</strong>。</p>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">而舍费尔告诉你：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">你自己也要主动出击，把该存的存起来、该买的保险买齐、该规划的投资规划好。</strong></p>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">舍费尔最著名的观点之一是：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">复利是世界的第八大奇迹。</strong></p>
<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">但复利有一个前提：<strong style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: inherit; color: #1aad19; font-weight: bold">时间。</strong></p>

<p style=" text-align: left; line-height: 1.75; font-family: -apple-system-font,BlinkMacSystemFont, Helvetica Neue, PingFang SC, Hiragino Sans GB , Microsoft YaHei UI , Microsoft YaHei ,Arial,sans-serif; font-size: 16px; margin: 1.5em 8px; letter-spacing: 0.1em; color: #333">他的公积金账户，现在已经有40多万了。每年利息6000多，相当于每个月多了500块“被动收入”。</p>
