SpaceX暴跌4000亿、创业板新高：从《随机漫步的傻瓜》看投资中的运气与风险

SpaceX市值单日蒸发4000亿美元，创业板却创下历史新高，智谱半年涨18倍——这些故事背后，藏着投资中最容易被忽视的真相：运气。塔勒布告诉我们，我们远远低估了随机性在成功中的作用。读懂这本书，你会在暴富神话面前保持清醒，在投资失败时保持从容。

昨天，A股创业板创下历史新高，成交额高达3.74万亿。与此同时，大洋彼岸的SpaceX股价暴跌16%，市值单日蒸发4000亿美元。

一个天堂，一个地狱。

如果你昨天同时持有创业板ETF和SpaceX，你可能会觉得自己是"人生赢家"——但塔勒布会冷冷地告诉你：你可能只是那只幸运的猴子。

这不是要打击你。而是《随机漫步的傻瓜》这本书最想告诉我们的：我们远远低估了运气在投资和人生中的作用。

今天，我想借这三个故事——SpaceX的暴跌、创业板的狂飙、智谱的18倍神话——和你聊聊这本书最精华的认知。

01 SpaceX的"过山车"：为什么马斯克也会跌？

先说SpaceX的故事，因为它太适合用来说明"随机性"了。

6月12日，SpaceX以135美元发行价登陆纳斯达克，首日暴涨19%至160.95美元。接下来的两个交易日更是疯狂，6月16日盘中触及225.64美元的历史高位，总市值一度逼近3万亿美元。

然后？然后就一路暴跌。

到6月23日，SpaceX连续第三个交易日下跌，累计跌幅超过30%，市值蒸发超过4000亿美元，跌破IPO首日收盘价。

这个剧本，熟悉吗？

历史上，每次超级IPO都会上演类似的戏码：散户疯狂涌入 → 股价短期暴涨 → 机构获利了结 → 散户站在高岗。

但问题来了：那些在高位买入的人，是能力不行，还是运气太差？

《随机漫步的傻瓜》告诉我们一个扎心的真相：有些人靠运气赚到的钱，会误以为是自己的能力。

他们会在高位继续加仓，因为他们相信"我之前的判断是对的"。

但塔勒布会说：你的"成功"可能只是那只幸运的猴子——连续抛了10次正面，你以为自己是掷硬币大师，实际上只是概率在发挥作用。

SpaceX的暴跌有几个原因：

第一，高利率杀估值。 美联储鹰派预期升温，2年期美债收益率上行至4.23%，高估值科技股首当其冲。SpaceX市销率超百倍，靠的是远期AI叙事，现在"远期"贴现值下降了。

第二，IPO动能枯竭。 分析师迈克·奥罗克直言：“初期看好资金已全部完成进场，市场买盘基本耗尽。”

第三，大规模发债。 SpaceX宣布将发行至少200亿美元公司债，用于AI业务扩张。市场担忧：债务增加意味着风险增加。

但问题是：这些原因，你能提前预判吗？

如果你在6月16日买入SpaceX，你可能是这么想的：

“马斯克是天才，SpaceX是颠覆性创新，美股科技股长期看好……”

这些理由都对。但问题是——你买入的时机对吗？

塔勒布说，判断一个决策好不好，不能只看已经发生的结果，还要考虑"原本可能发生的其他结果"。

买入后暴涨 → 你是"有眼光的投资者"？

买入后暴跌 → 你是"韭菜"？

但同样的决策、同样的分析、同样的理由，只是时间不同，结果就天差地别。你能说前者比后者更聪明吗？

02 创业板的狂欢：为什么"赚指数不赚个股"？

看完SpaceX，再看A股。

6月22日，创业板指大涨2.52%，收报4359.55点，创历史新高。两市成交额3.74万亿，创年内最高。

券商股集体涨停，有色金属全线飘红，保险板块暴拉……

但与此同时，2500只股票下跌，33只个股跌停。

这是典型的"赚指数不赚个股"行情。

指数涨了，个股却在跌。换句话说：如果你持有的是创业板指数基金，你赚钱了；但如果你持有的是个股，你可能还在亏钱。

这时候，问题来了：那些持有创业板赚到钱的人，是真的看懂了市场，还是刚好站在风口上？

塔勒布会说：我们往往只看到"幸存者"，而忽略了"失败者"。

比如，媒体会大肆报道：

  • “XX基金经理年化收益30%”
  • “XX投资者靠创业板实现财务自由”

但没人会告诉你：

  • 有多少创业板股票在这期间跌了50%甚至退市？
  • 有多少投资者因为追高创业板而亏损离场？

这就是塔勒布说的**“存活者偏差”**（Survivorship Bias）。

我们只看到了活下来的"幸运儿"，却忽略了那些因为失败而"消失"的样本。这导致我们严重高估了成功的概率。

就像那个经典的比喻：

如果让无限多的猴子投掷硬币，总有一只猴子能连续100次抛出正面。

这只猴子会认为自己天赋异禀，是"投硬币大师"，围观者也会为之著书立说。

但实际上，它只是**“随机漫步的傻瓜”**。

放到投资上：牛市里赚钱的投资者，就像那只幸运的猴子——他可能只是刚好在对的时机买了对的东西，而不是因为他真的"懂"市场。

03 智谱的18倍神话：是实力还是运气？

说完SpaceX和创业板，再看一个更极端的案例：智谱。

2026年1月8日，智谱在港交所上市，发行价116.2港元，首日市值约528亿港元。

到6月22日，智谱股价最高触及2980港元，市值突破1万亿港元。上市不到半年，涨幅超过18倍。

这是什么概念？

腾讯奋斗了二十多年才到4万亿港元。智谱用了180天就走完了腾讯二十分之一的里程。

你可以说这是"硬科技"的胜利，是中国AI实力被资本市场认可，是中国版Anthropic的稀缺性溢价……

这些理由都对。

但问题是：如果你在1月份买入智谱，你是基于什么判断的？

  • 你真的懂GLM-5.2的技术架构？
  • 你提前预判到了马斯克和唐杰的"隔空对话"会点燃市场情绪？
  • 你算准了美国对Anthropic的出口管制会让开发者转向国产大模型？

大概率不是。

你可能只是：

  • 刚好有闲钱想试试水
  • 听朋友推荐就买了一点
  • 看了某篇分析文章觉得有道理

然后，你稀里糊涂地赚了大钱。

塔勒布管这叫**“幸运的傻瓜”**——他们因为随机性获得了成功，却误将运气归因于自己的能力。

更可怕的是：一旦你把自己的成功归因于能力，你下一次做决策时就会更加自信，更加激进。

然后，当运气不再站在你这边时……

04 为什么"专业投资者"也会犯傻？

你可能会说：那些散户不懂行，追涨杀跌，被割韭菜是活该。那专业投资者呢？基金经理呢？他们总该有能力吧？

塔勒布用他作为期权交易员的亲身经历告诉我们：在华尔街，“专业"并不能让你免受随机性的愚弄。

他见过太多"明星交易员”：

  • 连续3年跑赢大盘，被媒体捧为"股神"
  • 第四年遇到黑天鹅，一次亏光前三年的所有收益
  • 第五年……他们已经不在这个市场了

但媒体只记住了他们辉煌的那三年，没人记得他们的溃败。

这就是金融市场的本质： 在高度随机的环境中，总有一小部分人能连续几年获利。但这不是因为他们掌握了可靠的赚钱秘诀——他们只是"幸运的猴子"。

更讽刺的是：当他们成功时，他们会把自己的策略写成书、开训练营、收咨询费。

但他们的"成功"可能纯粹是运气——一次随机波动，刚好让他们站在了风口上。

你学了他的"必胜法则"，结果……

05 火鸡悖论：为什么我们总被"过去"欺骗？

说完投资，再说说我们认知上的问题。

塔勒布在书中提到了一个经典比喻：火鸡悖论。

一只火鸡被农夫喂养了1000天。每天早上9点，农夫准时来投喂食物。

通过1000天的"数据观察"，火鸡得出了一个"真理"：“农夫 = 爱与食物的提供者”。

它甚至可以用复杂的数学模型来验证这个结论：

  • 相关系数 = 0.99
  • p值 < 0.001
  • 置信区间 = 99.9%

但感恩节那天，农夫来不是投喂的——是来宰杀的。

这个故事说的就是我们人类认知世界的基本方法：归纳法。

  • 看到1万只白天鹅 → “天鹅都是白的”
  • 连续100天赚钱 → “我的策略是有效的”
  • 房价连续20年上涨 → “买房永远不会亏”

塔勒布说：归纳法在"黑天鹅"面前不堪一击。

放到投资上：

  • 过去10年美国科技股年化收益15% → 你以为这是"正常"的
  • 2000年互联网泡沫破裂 → 你的"正常"世界观崩塌
  • 2022年科技股暴跌 → 你的"长期持有"信仰动摇

我们用过去预测未来，但未来往往和过去完全不同。

SpaceX的投资者们用"马斯克的神话"预测未来，但忘了：神话也是随机性的产物。

06 杠铃策略：普通人如何在不确定中存活？

说了这么多"随机性"的恐怖，你是不是有点绝望？

别急。塔勒布虽然戳破了"能力决定成败"的幻觉，但他也给出了应对之道：杠铃策略。

什么是杠铃策略？

一端极端保守，一端极度冒险，中间不要。

具体来说：

保守端：把大部分资金（比如85%-90%）放在几乎零风险的地方——国债、货币基金、存款。哪怕收益率只有3%-4%，你也亏不了。

冒险端：用小比例资金（比如10%-15%）去赌高赔率的机会——比如买入深度虚值期权、押注某只高风险股票。即使全亏，你也只是亏了10%-15%，不会伤筋动骨。

中间地带：不要。

为什么"中间地带"不要？

因为在塔勒布看来，中庸其实是最大的风险。

  • 中等收益、中等风险 → 你以为这是"稳健"
  • 但实际上，你承担了全部的下行风险，却放弃了全部的上行收益
  • 更重要的是：中等风险往往伴随着"温水煮青蛙"——你不会一下子亏光，但会慢慢被侵蚀

杠铃策略的核心逻辑是：让你的损失永远可控，让你的收益没有上限。

就像书里说的：

你的损失最多是投入的那一小部分，可一旦遇到黑天鹅机会，收益却是没有上限的。

放到个人成长上：

  • 保守端：做好本职工作，确保稳定的收入和生活
  • 冒险端：用业余时间尝试有可能改变人生的副业、学习新技能、结交不同领域的人

即使副业失败了，你还有主业。但如果成功了，你的人生可能就此改变。

07 真正的智慧：承认运气，保持谦逊

说了这么多，塔勒布最想告诉我们的，其实是一个人生哲学：

承认运气，保持谦逊。

当我们成功时，不要急着总结"成功法则"。

问问自己：这一次，有多少是能力，有多少是运气？

当别人成功时，不要急着模仿他的"方法论"。

先问问他：你成功了，你有没有想过你可能是那只幸运的猴子？

塔勒布说：最大的智慧是知道自己的边界在哪里。

你知道什么是自己可以控制的，什么是不能控制的。

你能控制的：自己的决策过程、自己的努力程度、自己的风险敞口

你不能控制的：市场的涨跌、运气的好坏、黑天鹅何时降临

专注于你能控制的，接受你不能控制的——这才是真正的"理性"。

08 结语：在这个疯狂的世界里，做一个清醒的人

SpaceX暴跌4000亿，创业板创下新高，智谱半年涨18倍……

这些故事每天都在上演。

有人暴富，有人破产，有人追悔莫及，有人意气风发。

如果我们只看结果，我们会得出结论：成功的人都是天才，失败的人都是傻瓜。

但塔勒布告诉我们：别被结果骗了。

成功的人，可能是那只幸运的猴子。他的"方法论"可能只是事后诸葛亮式的包装。

失败的人，可能只是运气不好。他的"错误决策"可能在另一个平行宇宙里大获成功。

真正的智慧，不是预测未来，而是承认未来的不可预测性。

就像《随机漫步的傻瓜》封面上那只若无其事走路的鸭子——水面之下，它的脚在拼命划动；但在水面上，它看起来从容淡定，仿佛一切尽在掌握。

我们都是那只鸭子。

区别只在于：有的人知道自己在划水，有的人不知道。

今天这篇文章，不是要你从此不投资、不冒险。

而是要你在下一次"成功"的时候，多问自己一句：这次，有多少是运气？

如果你能诚实地回答这个问题，你会发现：

保持谦逊，比任何"成功法则"都重要。

今日话题：

你有过"以为自己很厉害，后来发现只是运气好"的经历吗？

或者，你见过身边的人，因为一次成功就以为自己掌握了"财富密码"？
